# AC Writer - Jira Forge App

Generates Gherkin-format acceptance criteria using Rovo AI agents.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Login to Forge:
```bash
forge login
```

3. Register the app:
```bash
forge register
```

4. Deploy:
```bash
forge deploy
```

5. Install to your Jira site:
```bash
forge install
```

## Development

Run tunnel for live development:
```bash
forge tunnel
```

## Usage

1. Open any Jira issue
2. Find "AC Writer" in the right sidebar
3. Click "Generate Acceptance Criteria"
4. View generated Gherkin scenarios
